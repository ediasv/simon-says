# simon-says

Jogo desenvolvido para a disciplina "Técnicas de Programação"

## Build

1. Clone o repositório
2. Entre no diretório `simon-says`
3. Crie o diretório `build`
4. Entre no diretório build
5. Rode o comando `cmake ..`
6. Ainda no diretório `build` rode o comando `make`
7. O executável `exe` deve ter sido criado
